# 中关新园评论情感分析系统
> 文本挖掘、机器学习 系统可视化

## 系统核心功能

| 模块 | 技术实现 | 作用 |
| :--- | :--- | :--- |
| **全景可视化** | Pannellum.js | 360° 房间漫游 |
| **物体定位** | YOLOv8x | 自动识别房间内家具位置坐标 |
| **坐标转换** | opencv-python | 将 YOLO 识别出的 2D 像素坐标精确转换为 3D 全景图所需的角度 |
| **情感分析** | LLM (roBERTa) | 解析评论文本，判断评论情感 |
| **数据整合** | Flask | 实现评论提交、存储、查询 |

---

### 环境搭建与运行

- with `uv`, on `Windows`

```bash
cd backend
uv sync
uv run app.py
```
